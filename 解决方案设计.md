## 三、核心模块实现方案

### 3.1 模块基类（base.py）

**设计目的：** 统一模块接口，便于扩展和维护



### 3.2 去重模块（deduplication.py）

**算法流程：**

1. 阶段一：pHash 快速预筛选
   - 读取图片并缩放到 32x32 灰度图
   - 计算 pHash 值（默认 16x16 hash）
   - 使用汉明距离聚类，距离 <= 阈值的归为候选组
   - 每组保留第一张作为代表

2. 阶段二：SSIM 精细比对
   - 仅对候选组内的图片进行 SSIM 计算
   - 使用滑动窗口（默认 7x7）计算结构相似度
   - SSIM > 95% 标记为重复，保留第一张

**记录参数：** `phash`, `phash_hamming_distance`, `ssim_score`, `is_duplicate`, `duplicate_of`

### 3.3 去模糊模块（deblur.py）

**BREN 算法详解：**

BREN（Brenner's Reduced Energy Norm）是一种基于梯度的锐度评估方法：

```python
def compute_bren_sharpness(image: np.ndarray) -> Tuple[float, float]:
    """
    计算 BREN 锐度值和拉普拉斯方差

    BREN = sum(max(0, I(x+2,y) - I(x,y))^2) / (H * W)
    拉普拉斯方差 = var(Laplacian(image))
    """
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY) if len(image.shape) == 3 else image

    # BREN 计算
    shifted = np.roll(gray, -2, axis=1)
    diff = gray[:, :-2].astype(np.float64) - shifted[:, :-2].astype(np.float64)
    bren = np.sum(diff ** 2) / gray.size

    # 拉普拉斯方差
    laplacian = cv2.Laplacian(gray, cv2.CV_64F)
    lap_var = laplacian.var()

    return float(bren), float(lap_var)
```

**算法流程：**

1. 读取图片，可选缩放到指定尺寸（加速计算）
2. 转为灰度图
3. 计算 BREN 锐度值和拉普拉斯方差
4. 锐度低于阈值的图片标记为模糊

**记录参数：** `bren_sharpness`, `laplacian_variance`, `is_blurry`

### 3.4 异常检测模块（anomaly_detection.py）

**算法流程：**

1. 读取图片并转为 HSV 色彩空间
2. 计算亮度（V通道）统计：均值、标准差、最小值、最大值
3. 计算饱和度（S通道）统计：均值
4. 计算熵值（基于归一化灰度直方图）
5. 综合判断异常类型

**记录参数：** `brightness_mean`, `brightness_std`, `entropy`, `saturation_mean`, `anomaly_type`

## 四、并发处理与性能优化

### 4.1 多进程处理架构

```python
from concurrent.futures import ProcessPoolExecutor, as_completed
import multiprocessing

class ParallelProcessor:
    def __init__(self, config: Dict):
        self.num_workers = config.get('num_workers', 0) or multiprocessing.cpu_count()
        self.batch_size = config.get('batch_size', 100)

    def process_folder(self, folder_path: str, modules: List[BaseModule]) -> List[Dict]:
        """并行处理单个文件夹"""
        image_files = self._scan_images(folder_path)

        # 分批处理
        results = []
        for batch in self._chunk(image_files, self.batch_size):
            with ProcessPoolExecutor(max_workers=self.num_workers) as executor:
                futures = {
                    executor.submit(self._process_single, img, modules): img
                    for img in batch
                }
                for future in as_completed(futures):
                    try:
                        results.append(future.result())
                    except Exception as e:
                        logger.error(f"处理失败: {futures[future]}, 错误: {e}")

        return results
```

### 4.2 内存管理策略

```python
import gc
import psutil

class MemoryMonitor:
    def __init__(self, limit_mb: int = 2048):
        self.limit_mb = limit_mb

    def check_and_collect(self) -> bool:
        """检查内存使用，必要时触发垃圾回收"""
        current_mb = psutil.Process().memory_info().rss / 1024 / 1024
        if current_mb > self.limit_mb:
            gc.collect()
            return True
        return False

    def get_memory_usage(self) -> float:
        """获取当前内存使用量（MB）"""
        return psutil.Process().memory_info().rss / 1024 / 1024
```

### 4.3 SSIM 计算优化策略

**问题：** SSIM 两两比对复杂度为 O(n²)，大图片集效率低下

**优化方案：**

1. **pHash 预过滤：** 仅对 pHash 汉明距离 <= 阈值的图片对计算 SSIM
2. **分组聚类：** 先用 pHash 聚类，组内再计算 SSIM
3. **采样比对：** 对超大图片集，随机采样代表性图片进行比对

### 6.2 异常处理策略

| 异常类型 | 处理策略 | 是否中断 |
|---------|---------|---------|
| 损坏图片 | 记录日志，跳过该图片，继续处理 | 否 |
| 配置错误 | 记录日志，使用默认值或终止 | 可配置 |
| 内存超限 | 触发垃圾回收，降低批处理大小 | 否 |
| 模块异常 | 记录日志，跳过该模块，继续其他模块 | 否 |
| IO异常 | 记录日志，重试3次后跳过 | 否 |
| 未知异常 | 记录详细堆栈，跳过当前图片 | 否 |

def validate_image(file_path: str) -> Tuple[bool, str]:
    """
    验证图片是否有效

    返回：(是否有效, 错误信息)
    """
    try:
        with Image.open(file_path) as img:
            img.verify()
        with Image.open(file_path) as img:
            img.load()
        return True, ""
    except Exception as e:
        return False, str(e)

## 十、进度显示模块

```python
from tqdm import tqdm

class ProgressTracker:
    def __init__(self, total: int, desc: str = "处理中"):
        self.pbar = tqdm(total=total, desc=desc, unit="张")

    def update(self, n: int = 1, msg: str = ""):
        self.pbar.update(n)
        if msg:
            self.pbar.set_postfix_str(msg)

    def close(self):
        self.pbar.close()
```
